About this prototype

This form builder allows you to:

  • Create multi-page forms
  • Add various field types including text, radio buttons, checkboxes and more
  • Preview your form as you build it
  • Manage form data using browser storage
! Warning This is a prototype. Form data is stored in your browser's localStorage and will be cleared if you clear your browser data.
